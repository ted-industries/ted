# ted

minimal code editor for agents

---

### core
- [x] tree-sitter syntax highlighting
- [x] multi-pane buffer management
- [x] semantic theming system
- [x] high-density file tree (20px)
- [x] telemetry & behavior tracking
- [ ] detached window support

### git
- [x] visual commit graph & history
- [x] line-level blame (ghost text)
- [x] integrated status & repo management
- [ ] inline diff & hunk staging

### agents
- [x] tool-calling execution loop
- [x] action trace visibility (collapsible)
- [x] file-system tools (read, write, grep, list)
- [ ] multi-file planning & reasoning loops

### planned

- [ ] sandboxed execution environment
- [ ] lsp-driven code intelligence api
- [ ] multi-file planning & reasoning loops
- [ ] plugin system (wasm tools)
- [ ] remote development (ssh / containers)
